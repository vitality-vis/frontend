## Installation
1. Clone  or download the repository
2. `yarn install` (or `npm install` for npm)

### (Node.js, NPM) versions tested on

- (`v10.13.0`, `v6.4.1`)
- (`v12.4.0`, `v6.9.0`)

## Run

- `yarn run start-dev`
- Open `http://localhost:8080`
